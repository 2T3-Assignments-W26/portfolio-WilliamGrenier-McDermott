<--Title Git Cheat Sheet-->
**List of Useful Git Commands**
1. `git config -- global user.name "will"`
   set your Git username
2. `git config --global user.email "example@exampl.com"`
   Will set your Git email
3. `git init`
   Will initialize git in local repo
4. `git clone<url>`
   Will clone a repository using a link to it
5. `git add .`
   Will add all modified files to staging area
6. `git status`
   Gets status of current staged and commited files
7. `git diff`
   Will show difference between commits and current stage vs commit
8. `git commit -m`
   will commit staged files in a new commit of the project **snapshot**
9. `git mv <file 1> <file 2>`
    moves file 1 to file 2 could rename
10. `git push/pull`
    savign and recieving code from remote repo
11. `git rm`
    removes file from internal folder **delete**
